# Design.md

# WeDigMark Design System

Status: Active Development

---

# Purpose

This document defines the visual direction of the website.

It is intended to preserve consistency while allowing the design system to evolve as the website matures.

Design decisions documented here should reflect the current implementation, not future assumptions.

---

# Current Stage

The website is currently in the structural phase.

The primary objective is to establish:

- Information hierarchy
- Navigation
- User flow
- Content structure
- Component architecture

Visual refinement will occur after the website structure is complete.

---

# Design Goals

The website should feel:

- Premium
- Intentional
- Minimal
- Trustworthy
- Strategy-first

Every visual decision should support communication before aesthetics.

---

# Current Design Language

The current implementation uses:

- Dark theme
- Orange accent colour
- Minimal interface
- Large typography
- Generous whitespace
- Card-based layouts

These are implementation details and may evolve.

---

# Future Evolution

Future work may include:

- Motion design
- Improved visual hierarchy
- Better typography scale
- More refined spacing
- Advanced interactions
- Improved responsiveness
- Illustration or imagery systems
- Design tokens
- Component variants

These decisions should be documented here once finalized.

---

# Ownership

Visual design decisions are owned by the design lead.

Implementation should follow approved designs as closely as practical.

This document should evolve alongside the design system.
